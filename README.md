VIM config file
===============


## File list

* init.sh: install script for the VIM config (Linux only).
* _vimrc: the VIM config file, it will be renamed to .vimrc by init.sh
* UltiSnips: my code snippets Using UltiSnips format

## Install

1. Clone this repo
2. Run init.sh
3. Open vim, run :PlugInstall
